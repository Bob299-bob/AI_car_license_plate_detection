
# 🚗 AI Car License Plate Detection (YOLO + OpenCV)

## 🧠 Overview

This project is a **Computer Vision-based Car License Plate Detection system** that uses **YOLO (You Only Look Once)** for object detection and **OpenCV** for image processing.
It detects license plates from car images and displays results through a simple **Streamlit web application**.

---

## 🎯 Features

* Detects cars and license plates using **YOLO model**
* Crops and highlights detected license plates
* Uses **OpenCV for preprocessing and image handling**
* Simple **Streamlit web interface**
* Supports image-based inference

---

## 🛠️ Tech Stack

* Python 🐍
* YOLO (Object Detection Model)
* OpenCV (Image Processing)
* Streamlit (Web App)
* NumPy

---

## 🚀 How It Works

1. Input image is uploaded
2. YOLO model detects license plate region
3. OpenCV processes and crops detected area
4. Result is displayed on Streamlit UI

---


## ⚠️ Challenges Faced

* Understanding YOLO annotation format initially
* Dataset quality and inconsistency
* Detection issues in low light / angled images
* Integrating YOLO output with OpenCV pipeline
* Deployment issues on Streamlit

---

## 🔄 Improvements Made

* Cleaned and structured dataset
* Improved preprocessing using OpenCV
* Fine-tuned YOLO model for better accuracy
